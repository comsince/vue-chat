# 关于界面功能说明

## 功能概述

本次更新为即时通讯应用增加了关于界面功能，用户可以通过侧边栏底部的关于按钮查看软件版本、作者和编译时间等信息。

## 实现内容

### 1. 新增组件

**文件位置**: `src/components/menu/about.vue`

- 创建了独立的关于对话框组件
- 显示应用名称、版本号、作者、编译时间和描述信息
- 采用模态对话框设计，具有良好的用户体验
- 支持点击背景区域或关闭按钮关闭对话框

### 2. 侧边栏增强

**文件位置**: `src/components/mycard/mycard.vue`

- 在侧边栏底部添加了关于按钮
- 使用现有图标系统（icon-dkw_xiaoxi）
- 按钮位于退出按钮左侧，布局合理

### 3. 构建配置优化

**修改文件**:
- `config/prod.env.js` - 生产环境配置
- `config/dev.env.js` - 开发环境配置

增加了以下环境变量：
- `APP_VERSION`: 从package.json自动读取版本号
- `BUILD_TIME`: 自动生成编译时间（中文格式）

### 4. 显示信息

关于界面显示以下信息：
- **应用名称**: WeChat Web
- **版本号**: 1.0.14（从package.json读取）
- **作者**: comsince  
- **编译时间**: 实时生成的编译时间
- **描述**: 基于unverse-push的开源即时通讯web客户端

## 使用方法

1. 启动应用后，在左侧边栏底部可以看到关于按钮（消息图标）
2. 点击关于按钮，弹出关于对话框
3. 查看完毕后，可以点击"确定"按钮或对话框外部区域关闭

## 技术特点

1. **响应式设计**: 关于对话框支持移动端适配
2. **环境变量注入**: 版本和编译时间通过webpack在构建时注入
3. **组件化开发**: 关于功能作为独立组件，便于维护
4. **用户体验**: 采用现代化的UI设计，符合应用整体风格

## 构建和部署

构建命令：
```bash
npm run build
```

开发模式：
```bash
npm run dev
```

构建完成后，关于界面将显示正确的版本信息和编译时间。

## 注意事项

1. 编译时间在每次构建时自动更新
2. 版本号与package.json保持同步
3. 关于按钮使用现有图标库，无需额外资源
4. 对话框采用固定定位，确保在所有页面状态下正常显示
